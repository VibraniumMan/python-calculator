# 🧮 Python Calculator

A simple desktop calculator built with Python using the built-in Tkinter library.

## Features
- Addition, subtraction, multiplication, and division
- Percentage and positive/negative toggle
- Keyboard support (type numbers and operators directly)
- Backspace to delete last character
- Dark themed UI

## How to Run
Make sure Python is installed, then run:
    python calculator.py

## Built With
- Python 3
- Tkinter (built into Python, no installation needed)
